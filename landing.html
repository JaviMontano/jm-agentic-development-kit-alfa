Cómo funciona

Describes lo que necesitas

«Necesito una app de tareas para equipos pequeños con colaboración en tiempo real.»

Pristino compone una tríada

El orquestador selecciona 3 agentes: un Lead (especialista del dominio), un Support (seguridad, accesibilidad) y un Guardian (calidad). Trabajan en secuencia.

Cada agente ejecuta su rol

Lead produce el entregable. Support revisa edge cases y vulnerabilidades. Guardian valida contra la Constitución y las 4 puertas de calidad.

Recibes un proyecto real

Código funcional, probado, accesible, seguro, desplegado en URL en vivo. Con decisiones documentadas y trazables.

«La brecha entre aspirar y lograr se cierra con método.»

Cómo empezar

Instala Claude Code

Descárgalo en claude.ai/claude-code. Es el entorno recomendado. También funciona en Cursor, VS Code, Gemini, Windsurf, Codex.

Clona e instala

git clone https://github.com/JaviMontano/jm-agentic-development-kit.git
claude plugin add ./jm-agentic-development-kit

Arranca

/jm-adk:init configura workspace automático + Constitución + tríada.
/jm-adk:onboarding te guía paso a paso.
/jm-adk:demo te muestra el kit en acción.
