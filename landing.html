Qué es esto

JM-ADK es un kit que convierte tu asistente de IA en un equipo completo de desarrollo. Tú describes lo que quieres. El kit analiza, diseña, construye, prueba y despliega.

256

Habilidades

Desde crear un sitio web hasta prospectar clientes, diseñar APIs, o escribir propuestas comerciales.

256

Agentes

Cada uno es experto en algo: seguridad, diseño, Firebase, marketing, contenido, calidad.

1024

Componentes

Habilidades + agentes + comandos + prompts. Todos gobernados por una Constitución de 18 principios.

Cómo funciona

Tres pasos. Tú describes. El kit construye.

Describe lo que quieres

"Necesito una app de tareas para equipos pequeños con colaboración en tiempo real."

El kit arma un equipo

Pristino (el orquestador) selecciona 3 agentes: uno que lidera, uno que revisa, uno que valida calidad. Trabajan en secuencia, como una línea de ensamblaje.

Recibes un proyecto real

Código funcional, probado, accesible, seguro. Desplegado en una URL en vivo. Con documentación y decisiones trazables.

"La brecha entre aspirar y lograr, se cierra con método."

Para quién es

Emprendedores

Tienes la idea pero no el equipo técnico. El kit la convierte en producto funcional.

Desarrolladores

Ya sabes código. El kit te da gobernanza, calidad, y velocidad — sin sacrificar rigor.

Consultores

Necesitas analizar, proponer, y entregar. El kit tiene el pipeline completo: discovery → desarrollo → deploy.

Equipos

256 agentes especializados que trabajan con la misma Constitución. Consistencia sin fricción.

Qué puedes hacer

Crear sitios web

Portafolios, landing pages, blogs. Lighthouse 98+, accesible, SEO-ready.

Construir apps

Task managers, dashboards, e-commerce. Firebase + auth + real-time.

Prospectar clientes

Investigación de mercado, dossiers de empresas, secuencias de outreach B2B.

Diseñar propuestas

Workshops, presupuestos, roadmaps, business cases. Todo con tu marca.

Analizar proyectos

AS-IS, flujos de negocio, stakeholders, factibilidad, estimación de costos.

Mejorar tu CV

Análisis de perfil, reescritura con impacto, formato HTML de marca.

Cómo empezar

Instala Claude Code

Descárgalo en claude.ai/claude-code. Es el entorno donde corre el kit.

Clona e instala el kit

git clone https://github.com/JaviMontano/jm-agentic-development-kit.git
claude plugin add ./jm-agentic-development-kit

Inicia tu primer proyecto

/jm-adk:onboarding — Te guía paso a paso. O ve directo: /jm-adk:demo
