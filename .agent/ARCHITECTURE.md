## Quality Gates

- **G0**: Security — no exposed secrets/keys
- **G1**: Evidence — all claims tagged
- **G2**: Lighthouse — Performance > 90, Accessibility > 95
- **G3**: Deployment — works on Hostinger without server-side runtime
