## Descripción

**JM Agentic Development Kit Alfa** es un kit completo de desarrollo agéntico. Incluye 5 plugins integrados (MAO, SA, PM-APEX, SDD, PQA), workspace automático, y un pipeline de especificación a producción. Describe lo que quieres, el kit analiza, diseña, construye, prueba y despliega con trazabilidad completa.

> **1036 componentes** · **264 skills** · **256 agentes** · **5 plugins integrados**

---

## Instalación

```bash
git clone https://github.com/JaviMontano/jm-adk-alfa.git && cd jm-adk-alfa
```

---

## Características Principales

| Característica | Descripción |
| --- | --- |
| **Pipeline completo** | spec → build → test → deploy en un solo flujo |
| **5 plugins integrados** | MAO, Sovereign Architect, PM-APEX, SDD, PQA |
| **256 agentes especializados** | Cada agente resuelve un dominio específico del ciclo de vida |
| **Zero-Hallucination Protocol** | Verificación de hechos en cada paso del pipeline |
| **Quality Gates G0-G3** | Puertas de calidad progresivas antes de cada entrega |
| **Patrón MOAT** | Meta-Orchestration Agentic Topology para coordinación multi-agente |
